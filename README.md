# Nvidia Shield Netflix Button Remap
An Android TV application that intercepts the Netflix button on Nvidia Shield remotes, allowing you to:

- **Disable the button** completely (prevents accidental launches)
- **Remap to another app** (YouTube, Plex, Kodi, etc.)
- **Keep Netflix working** - launch it manually whenever you want

No root required. Simple configuration UI. Works on all Nvidia Shield models (2015, 2017, 2019, Pro).
